A custom checkbox built on `Pressable`. The check mark is a CSS-rotated rectangle, so the component has no SVG dependency.

<ComponentPreview slug="components/checkbox" />

## Install

```sh
npx stylesheet-ui add checkbox
```

## Usage

```tsx
import { useState } from "react";
import { Checkbox } from "@/components/ui/checkbox";

const [accepted, setAccepted] = useState(false);

<Checkbox
  checked={accepted}
  onCheckedChange={setAccepted}
  label="I accept the terms"
/>
```

## Props

| Prop              | Type                            | Default | Description |
|-------------------|---------------------------------|---------|-------------|
| `checked`         | `boolean`                       | —       | Controlled checked state |
| `onCheckedChange` | `(next: boolean) => void`       | —       | Fired on press |
| `label`           | `ReactNode`                     | —       | Optional label rendered to the right |
| `size`            | `"sm" \| "md" \| "lg"`            | `"md"`  | 16 / 20 / 24 px |
| `disabled`        | `boolean`                       | `false` | Mutes opacity, disables press |
