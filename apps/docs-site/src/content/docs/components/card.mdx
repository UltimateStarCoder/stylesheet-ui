A container with a configurable surface (`default`, `muted`, `outline`), padding scale, and optional press handling.

<ComponentPreview slug="components/card" />

## Install

```sh
npx stylesheet-ui add card
```

## Usage

```tsx
import { Card } from "@/components/ui/card";

<Card>
  <Text>Default surface with shadow.</Text>
</Card>

<Card surface="muted" padding="lg">
  <Text>Tinted background, no shadow, generous padding.</Text>
</Card>

<Card pressable onPress={() => {}}>
  <Text>Press dims opacity.</Text>
</Card>
```

## Props

| Prop        | Type                                                | Default     | Description |
|-------------|-----------------------------------------------------|-------------|-------------|
| `surface`   | `"default" \| "muted" \| "outline"`                  | `"default"` | Visual treatment |
| `padding`   | `"none" \| "sm" \| "md" \| "lg"`                     | `"md"`      | Interior spacing |
| `pressable` | `boolean`                                           | `false`     | Renders a `Pressable` instead of a `View` |
| `onPress`   | `PressableProps["onPress"]`                         | —           | Press handler (requires `pressable`) |
| `style`     | `StyleProp<ViewStyle>`                              | —           | Extra style |
